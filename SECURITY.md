# Security Policy / セキュリティポリシー

## Supported Versions / サポート対象のバージョン

We currently only support the versions of the scripts for security updates. 
現在、セキュリティアップデートのサポート対象は以下のバージョンのみとなります。

| Component (コンポーネント) | Version (バージョン) | Supported (サポート) |
| :--- | :--- | :--- |
| **Main Script (macOS)** | m1.11 | :white_check_mark: |
| **Main Script (Windows)** | w1.0 | :white_check_mark: |
| **Installer (macOS)** | m1.6 | :white_check_mark: |
| **Installer (Windows)** | w1.1 | :white_check_mark: |
| All older versions (上記以前の古いバージョン) | - | :x: |

## Reporting a Vulnerability / 脆弱性の報告について

If you discover a security vulnerability within this project, please **DO NOT** create a public GitHub Issue.
本プロジェクト（パスワードの保存や送信処理など）においてセキュリティ上の脆弱性を発見した場合は、**公開のIssueには書き込まないでください。**

Please report it by contacting us directly via email:
代わりに、以下のメールアドレス宛に直接ご報告をお願いいたします。

* **Email**: [s@shintaromurase.com]

We will review all reports and respond as quickly as possible.
いただいた報告は速やかに確認し、修正に向けた対応をご連絡いたします。
